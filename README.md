# Blue Ocean Login

Page de connexion responsive pour l'application **Blue Ocean**, développée en HTML et CSS pur.

##  Description

Interface de connexion moderne et épurée comprenant :
- Un formulaire avec champs Email et Mot de passe
- Une connexion via réseaux sociaux (Google, Facebook, GitHub)
- Un design responsive adapté mobile, tablette et desktop
- Une mise en page deux colonnes avec illustration à gauche

## 🖥️ Aperçu

| Desktop | Mobile |
|--------|--------|
| Layout deux colonnes | Formulaire plein écran |
| Image + Formulaire |

##  Responsive

| Breakpoint | Comportement |
|-----------|-------------|
| `< 640px` | Image masquée, formulaire plein écran |
| `641px - 900px` | Carte compacte, image réduite |
| `> 900px` | Layout deux colonnes complet |
